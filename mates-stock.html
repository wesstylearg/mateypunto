🧉 Mate y Punto
Para guardar tus datos en la nube necesitás conectar tu cuenta de GitHub una sola vez. Los datos se almacenan en un Gist privado tuyo.
1
Entrá a GitHub → Settings → Tokens y creá un token con el permiso gist habilitado (classic token). Copialo antes de cerrar la página.
2
Pegá el token acá abajo. Se guarda solo en tu navegador y nunca se envía a ningún servidor externo, solo a la API de GitHub.
